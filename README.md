## Quick start

Run from command line

```
git clone https://github.com/vicdata4/lit-course

cd lit-course
```

Install dependencies
```
npm install
```

Run application

```
npm run dev
```

Run tests

```
npm run test
```

Run ESLint

```
npm run lint
```

Run ESLint Autofix

```
npm run lint:fix
```

Run storybook

```
npm run storybook
```
